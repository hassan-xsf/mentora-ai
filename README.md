# Mentora AI

An AI-powered career learning platform that helps students discover their ideal tech career, generate personalised learning roadmaps, practise coding, and track progress — all driven by Google Gemini.

---

## Project Overview

AI Student Platform guides students from "I don't know what to study" to a structured, personalised learning path. A student completes a 10-question career assessment, receives AI-ranked career recommendations, generates a full multi-section learning roadmap for their chosen career, works through nodes and tasks, takes milestone tests to unlock new sections, practises coding with AI-generated challenges, and tracks everything on a live progress dashboard.

---

## Features

| Feature | Description |
|---|---|
| **Career Assessment** | 10-question quiz (single/multi-choice, scale) → AI matches to 5 tech careers with fit scores and salary data |
| **AI Roadmap Generation** | One click generates a 3-section (Foundations → Intermediate → Advanced) roadmap with nodes, resources, and tasks |
| **Section Gating** | Sections unlock only after passing the milestone test (≥ 70%) for the previous section |
| **AI Milestone Tests** | 5 MCQ questions generated by AI per section; answers auto-graded |
| **AI Coding Challenges** | Choose language, topic, and difficulty; AI writes the problem and starter code; submitting triggers AI evaluation with scores and feedback |
| **AI Tutor Chat** | Educational-only streaming chat assistant with persistent history; context-aware when inside a roadmap node |
| **XP & Gamification** | XP awarded on challenge completion, level calculated from XP, streak counter, badge unlocks |
| **Progress Dashboard** | Activity charts (daily/weekly), stat cards (nodes, challenges, tests, avg score), badge showcase |
| **Mobile Responsive** | Full mobile navigation drawer, responsive layouts across all pages |

## Installation Guide

### Prerequisites

- Node.js 18+
- A [Supabase](https://supabase.com) project
- A [Google AI Studio](https://aistudio.google.com) API key

### 1. Clone and install dependencies

```bash
git clone <repo-url>
cd my-project
npm install
```

### 2. Set up environment variables

Create a `.env.local` file in the project root:

```env
# Supabase
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY=your_supabase_anon_key
SUPABASE_SECRET_KEY=your_supabase_service_role_key

# Google Gemini
GEMINI_API_KEY=your_gemini_api_key

# App
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

### 3. Set up the database

Open your Supabase project → **SQL Editor** and run the full setup file:

```
supabase/migrations/000_full_setup.sql
```

This single file creates all tables, enables Row Level Security, adds RLS policies, creates indexes, sets up the `handle_new_user` auth trigger, and seeds career and assessment data.

> **Important**: `SUPABASE_SECRET_KEY` (service role key) is required for roadmap generation — it bypasses RLS when writing nodes and resources server-side. Never expose this key to the browser.

### 4. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## API Setup Instructions

### Supabase

1. Go to [supabase.com](https://supabase.com) → create a project
2. Navigate to **Project Settings → API**
3. Copy:
   - **Project URL** → `NEXT_PUBLIC_SUPABASE_URL`
   - **anon / public key** → `NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY`
   - **service_role / secret key** → `SUPABASE_SECRET_KEY`
4. Navigate to **Authentication → URL Configuration** and add `http://localhost:3000` to the allowed redirect URLs

### Google Gemini

1. Go to [Google AI Studio](https://aistudio.google.com)
2. Click **Get API key** → **Create API key**
3. Copy the key → `GEMINI_API_KEY`

The platform uses `gemini-2.0-flash-lite` for all AI calls (career suggestions, roadmap generation, challenge creation, milestone test generation, code evaluation, and chat).

## Deployment Process

### Deploy to Vercel (recommended)

1. Push the repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project** → import the repo
3. In **Environment Variables**, add all five variables from the table above
4. Set **Framework Preset** to `Next.js`
5. Click **Deploy**

After deploying:
- Update `NEXT_PUBLIC_SITE_URL` to your Vercel production URL
- In Supabase → **Authentication → URL Configuration**, add your production URL to allowed redirect URLs

### Build locally

```bash
npm run build
npm start
```

## Project Structure

```

  app/
    (auth)/           # sign-in, sign-up pages
    (dashboard)/      # dashboard, assessment, roadmap, practice, progress

  lib/
    auth/             # session utilities
    db/               # database access functions (roadmaps, progress, practice…)
    gamification/     # XP calculation, badge logic, streak handling
    openai/           # Gemini client + streaming helpers
    supabase/         # server client, browser client, admin (service-role) client
  types/              # domain types + generated database types
supabase/
  migrations/         # SQL — 000_full_setup.sql is the authoritative single file
AI.md                 # Full documentation of every AI call site and prompt
```
